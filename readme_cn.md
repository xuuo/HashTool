# HashTool

[English](README.md)

HashTool 是一个基于 **VB.NET + WPF** 开发的轻量级 **Windows 桌面文件校验工具**，用于计算并校验文件的 **MD5** 和 **SHA256** 值，帮助你快速验证文件完整性。

本项目专注于**文件校验**场景，核心特点是支持**多个文件的并行哈希计算**，能够在面对大量文件时显著缩短整体校验所需时间。界面简洁、使用直接，适合作为日常工具或 WPF / VB.NET 学习示例。

## ✨ 功能特性

- 支持 **文件哈希校验**（不包含文本哈希功能）
- 支持 **MD5** / **SHA256** 算法
- 支持 **文件拖拽** 操作，直接将文件拖入程序即可开始校验
- 支持 **批量文件哈希计算**
- 支持 **多个文件并行校验**，充分利用多核 CPU，在大量文件场景下显著提升哈希计算效率
- 自动根据本机 **CPU 核心数**匹配并行线程数
- 支持 **手动调整线程数量**，在性能与资源占用之间灵活取舍
- 支持 **校验结果导出**，便于保存与后续比对
- 支持 **一键复制校验结果**，方便快速使用
- 简洁直观的 WPF 图形界面
- 原生 Windows 桌面应用体验
- 项目结构清晰，便于阅读与二次开发

## 🚀 使用方法

### 1️⃣ 克隆仓库

```bash
git clone https://github.com/luoji12/HashTool.git
```

### 2️⃣ 使用 Visual Studio 打开项目

- 推荐使用 **Visual Studio 2019 及以上版本**
- 确保已安装 .NET 桌面开发相关工作负载

### 3️⃣ 编译并运行

- 编译解决方案并运行程序
- 通过 **文件选择** 或 **拖拽文件到窗口** 的方式添加待校验文件
- 支持一次添加多个文件，程序将自动进行批量处理
- 所有文件的 MD5 / SHA256 将以并行方式计算
- 程序会根据当前电脑 CPU 自动匹配合适的线程数量，也可由用户手动调节
- 在大量文件校验场景下，可显著缩短整体计算时间
- 与官方或期望的哈希值进行对比即可完成校验
